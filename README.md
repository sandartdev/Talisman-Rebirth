# Talisman Rebirth

Bem-vindo ao **Talisman Rebirth** - seu portal pessoal para descoberta e transformação!

## 🌟 Sobre

Este é um website moderno e responsivo construído com HTML, CSS e JavaScript. Ele apresenta:

- ✨ Design elegante e profissional
- 📱 Responsivo em todos os dispositivos
- 🎨 Gradientes modernos e animações suaves
- 📧 Formulário de contato funcional
- 🔗 Navegação suave

## 📁 Estrutura do Projeto

```
talismanrebirth/
├── index.html       # Página principal
├── style.css        # Estilos CSS
├── script.js        # Funcionalidades JavaScript
└── README.md        # Este arquivo
```

## 🚀 Como Usar

### Localmente
1. Clone o repositório:
```bash
git clone https://github.com/diegoresc/talismanrebirth.git
```

2. Navegue até a pasta:
```bash
cd talismanrebirth
```

3. Abra o arquivo `index.html` no seu navegador:
```bash
open index.html
# ou
start index.html
# ou simplesmente clique duas vezes em index.html
```

### Online
Você pode fazer deploy usando:
- **GitHub Pages** (gratuito)
- **Vercel**
- **Netlify**
- **Heroku**

## 🎨 Personalização

### Alterar Cores
No arquivo `style.css`, procure por:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

E substitua pelos códigos de cor de sua preferência.

### Adicionar Conteúdo
Edite o `index.html` para adicionar:
- Mais seções
- Suas informações pessoais
- Links para seus projetos
- Redes sociais

### Melhorar Funcionalidades
Expanda o `script.js` para adicionar:
- Validação mais robusta do formulário
- Efeitos de animação
- Integração com backend
